知识点

  1. 因式分解

    • 基本：提公因式

    • 基本：公式逆用

    • 基本：十字相乘

  • 拓展：长除法/解方程法
    建议看因式分解 - 基础篇

  1. 一元二次方程、二次函数和不等式

    • 初中知识：省略

    • 拓展：高次不等式的解法 “穿针引线” ( $T(x) \gt 0 \Leftrightarrow sgn(T(x)) = 1$ )

      1. 分解因式 (求出$T(x)=0$ 时的n个根，从大到小排的第 $i$ 个根 (记为 $A_i$ ) 重复次数记录为 $B_i$ )

      2. 在数轴上依次标注出所有的根

      3. 确定 $x \gt A_1$ 时 $T(x)$ 的正负性

      4. $\forall x \in(A_k, A_{k-1}), y \in (A_{k+1}, A_k)$

        • 若$B_k$ 是偶数，那么 $sgn(T(y))=sgn(T(x))$

        • 否则 $sgn(T(y))=-sgn(T(x))$

        • 注：这便是所谓的 “奇穿偶不穿”

      5. 记录下所有满足 $sgn(T(x)) = 1$ 的 $x$ 即可

  2. 向量加法（物理常用）

    • 几何方法：三角形法则和平行四边形法则

    • 代数方法：从原点出发的向量 $\vec A = (a, b), \vec B = (m, n)$ ，他们的加法结果

    $$ \vec A + \vec B = (a+m, b+n) $$

物理

  1. 基本概念

    • 质点

      • 忽略物体本身性质，只考虑其整体运动情况

      • 火箭着陆等涉及到物体本身性质的情况不能用质点模型考虑

    • 位移

      • 从起点指向终点的有向线段 ( 向量 )

      • 不考虑中间线路长度 (操场跑一圈回到原地，整体位移是 0 )

    • 自由落体

      • 没有空气阻力的情况下，下落速度的增量都是一样的，计算公式 (装X时间到~ )：

      $$ h = \frac{1}{2}gt^2 + v_0t = \int v,dt\\ v = gt+v_0 $$

  2. 学习指南

    • 会建立模型：要找出物理对象间的关系，联系的方式，从形象到抽象
