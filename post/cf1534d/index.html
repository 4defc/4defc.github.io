MultiSet 题解

Aug 28, 2023

题面：

Multiset - 洛谷

给定一个数组$a_n$ ( $1\le a_i \le n$ ) 你需要保证数组始终是升序的 ( 其实就是可重集 )

有$q$次询问

每次询问给出一个数$x$

  • 若$x \lt 0$，则删除$a[-x]$。如果此时$n \lt -x$，则忽略本次操作

  • 否则，向数组中插入$x$

最后，输出数组中任意一个数，若数组为空则输出0

输入：

第一行两个数$n, q$

第二行$n$个数，是数组$a[1..n]$的元素

第三行$q$个数，每个数都是一个询问

样例 1

输入
5 5
1 2 3 4 5
-1 -1 -1 -1 -1
输出
0
解释

本样例中，数组的变化为 $[1,2,3,4,5]=>[2,3,4,5]=>[3,4,5]=>…=>[]$

样例 2

输入
5 5
1 2 3 4 5
-5 -1 -3 -1 -10
输出
3

最后一个询问：因为不存在，所以忽略

思路

令 $b_i$ 表示第 $i$ 个数出现了几次

二分答案，找到第k个数，满足 $\sum b[1..k] \ge -x$

$b_k$减1即可

可以说是权值树状数组的模板了

AC代码

#include<iostream>
#define ex constexpr

#define for1(i, b) for(int i = 1, t; i <= (b); ++i)
namespace {

}
/*
  思路其实很简单
  因为a[n] 里的东西都是1~n
  令b[i]=(a[]中出现了多少个i)
  找到第一个满足b[1..k]>=x的k即可删除(二分答案)
  用树状数组维护b
 */


int n;
intex MAXN = 1e6+5;

// 树状数组部分
// Begin
int tree_arr[MAXN];
inline lowbit(int x) {
	return x & -x;
}

void add(int i, int x) {
	for(; i<=n; i += lowbit(i))
		tree_arr[i] += x;
}

int get(int i) {
	int ans = 0;
	for(; i > 0; i -= lowbit(i)) 
		ans += tree_arr[i];
	return ans;
}

// End

int main () {
	std::ios::sync_with_stdio(0);
	cin.tie(0), cout.tie(0);

			while(left < right) {
				intc mid = left + right >> 1;
				if(get(mid) < t) 
					left = mid + 1;
				else right = mid;
			}
			add(left, -1);
		}
